* ANOVA
 - ttest와 유사하지만 특정 2 그룹만 활용하지만
   ANOVA의 경우 3개 이상의 그룹을 비교하고 싶은 경우 사용

 - 활용 함수
   : f_oneway(전체 그룹 값)

 - TIP
  lista=[np_data_2[:, x] for x in range(12)]
  stats.f_oneway(*lista)
  : *를 리스트 앞에 붙이게 되면 []를 벗겨낸 값을 의미, 즉 for, if로 생성된 리스트의 경우 for, if문에 적용됨



* 선형 대수

 - 벡터(vector)
  : 방향과 크기를 가지는 값
 - 스칼라(scala)
  : 크기만 가지는 값

 - value
  scalar ---> vector ---> matrix ---> tensor
    1D           2D            3D            4D+

 - 거리 산출 방식
  * 유클리드형(euclidean) distance
  * 멘하턴형(manhattan) distance
  * 민코우스키형(minkowski) distance  (전체 포괄형)

 - 벡터를 스칼라 값으로 표현하는 방법(Norm)
  : L1 norm = manhattan dist  = 절대값의 합
  : L2 norm = euclidean dist = 각 제곱의 합의 루트

 - SVD(Singular Value Decomposition)
  : 특이값 분해
  : 소인수분해 기법
  : 항상 3개의 matrix로 나온다
  : ex> np.linalg.svd(np_data)

 - PCA(Principal Component Analysis)
  : 주성분 분석
  : demension reduction(차원 축소) 기법
  : feature(dimension) reduction => pca 
    (계산을 통해 유사한 것을 통합해 차원 축소)
                             vs
    feature(dimension) selection 
    (임의 선택으로 차원 축소)
